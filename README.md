# 📊 Proyectos Pequeños de Data Science

## 🗂️ ¿Qué vas a encontrar aquí?

Proyectos de pequeña escala para aprender y mejorar en el mundo de los datos!

* **Para visualizar:** Power BI, Matplotlib y Seaborn.
